Comandos Básicos HTML

Questionário 1: Explique qual é a função de um servidor webem um ambiente de hospedagem e cite pelo menos um exemplo.

Resposta: O servidor web é responsável por receber as requisições dos navegadores, processá-las e entregar o conteúdo do site (como páginas HTML e imagens). Ele garante que o site seja acessado corretamente pelos usuários. Um exemplo popular de servidor web é o Apache HTTP Server.
Questionário 2: Qual a principal característica do NGINX e por que ele é muito utilizado em projetos de alto desempenho?

Resposta: A principal característica do NGINX é sua arquitetura assíncrona, que permite lidar com muitas conexões simultâneas de maneira eficiente. Ele é muito utilizado em projetos de alto desempenho por ser leve, rápido e ideal para balanceamento de carga e serviço de conteúdo estático.
Questionário 3: Diferencie os protocolos HTTP e HTTPS. Por que o HTTPS é recomendado para aplicações modernas?

Resposta: HTTP não tem criptografia, já HTTPS usa SSL/TLS para segurança. O HTTPS é recomendado para proteger dados sensíveis.
Questionário 4: Qual é a função das portas em uma rede de computadores e qual a importância da porta 443?

Resposta: Portas identificam serviços na rede. A porta 443 é usada para HTTPS, garantindo comunicação segura.
Questionário 5: Defina o conceito de endereço IP e explique sua importância em redes de computadores.

Resposta: O IP é o identificador único de um dispositivo na rede, essencial para comunicação entre eles.
Questionário 6: Explique o que é hospedagem VPS e em quais situações ela é mais indicada.

Resposta: VPS oferece recursos dedicados em um servidor compartilhado, ideal para sites com tráfego alto e necessidade de personalização.
Questionário 7: Descreva as vantagens do servidor web Caddy em relação a outros servidores tradicionais.

Resposta: O Caddy facilita a configuração de HTTPS e é simples de usar, com recursos de balanceamento de carga integrados.
Questionário 8: Qual servidor de aplicação é mais indicado para aplicações desenvolvidas na plataforma .NET (C#)? Explique o motivo.

Resposta: O IIS é o servidor ideal para .NET, pois é integrado ao Windows e otimiza a execução de aplicações ASP.NET.
Questionário 9: Descreva o funcionamento do modelo Cliente-Servidor no contexto de sistemas web.

Resposta: No modelo Cliente-Servidor, o cliente solicita e o servidor responde com conteúdo ou dados processados.
Questionário 10: Explique o que é o Apache Tomcat e para qual tipo de aplicação ele é mais indicado.

Resposta: O Tomcat é um servidor de aplicação Java, ideal para Java Servlets e JSP.
Questionário 11: Diferencie, com suas palavras, um servidor web de um servidor de aplicação, destacando suas funções principais.

Resposta: Servidor web serve conteúdo estático, enquanto servidor de aplicação processa lógica e gera conteúdo dinâmico.
Questionário 12: Defina o conceito de domínio na internet e sua relação com os endereços IP.

Resposta: Um domínio é um nome legível, mapeado para um endereço IP que identifica o servidor na rede.
Questionário 13: Quais as principais vantagens da hospedagem em nuvem em comparação com a hospedagem local?

Resposta: A nuvem oferece escalabilidade e flexibilidade, enquanto a local exige investimento em infraestrutura própria.
Questionário 14: Cite dois exemplos de servidores de aplicação e as linguagens/plataformas que eles suportam.

Resposta: Tomcat (Java), Node.js (JavaScript).
Questionário 15: Por que é essencial a utilização de HTTPS em sites e sistemas atualmente? Quais riscos o HTTP apresenta?

Resposta: HTTPS protege dados sensíveis e evita ataques. HTTP é vulnerável a interceptações.
Questionário 16: Dê um exemplo de situação onde o uso de hospedagem VPS seria mais vantajoso do que uma hospedagem compartilhada.

Resposta: VPS oferece mais controle e desempenho, ideal para sites com alto tráfego.
Questionário 17: Diferencie os conceitos de IP e porta em uma rede, explicando suas funções.

Resposta: IP identifica o dispositivo; porta define o serviço que processa a comunicação.
Questionário 18: Qual a importância de um servidor web conseguir lidar com conexões via HTTPS?

Resposta: Garante segurança na comunicação, protegendo dados dos usuários.
Questionário 19: Cite dois exemplos de servidores web e explique suas principais características.

Resposta: Apache: flexível, muitos módulos.
NGINX: rápido, leve, ideal para alto desempenho.
Questionário 20: Defina o que é uma hospedagem compartilhada e em quais casos ela costuma ser utilizada.

Resposta: Servidores compartilhados com vários sites. Ideal para sites pequenos e com baixo tráfego.
Questionário 21: Explique a função de um servidor de aplicação e a diferença em relação a servidores que apenas servem arquivos estáticos.

Resposta: Processa lógica de negócios e gera conteúdo dinâmico, ao contrário dos servidores que só servem arquivos estáticos.
Questionário 22: Cite um servidor de aplicação usado para projetos em Python e descreva como ele funciona.

Resposta: Gunicorn: servidor WSGI para executar aplicações Python.
Questionário 23: O que é o WildFly (ex-JBoss) e para qual tecnologia ele é indicado?

Resposta: Servidor de aplicações Java EE, ideal para aplicações corporativas Java.
Questionário 24: Qual a função da porta 21 em redes de computadores e para qual serviço ela é utilizada?

Resposta: A porta 21 é usada para o serviço FTP, que transfere arquivos entre dispositivos.
Questionário 25: Explique por que o servidor LiteSpeed é compatível com o Apache e quais as vantagens dessa compatibilidade.

Resposta: LiteSpeed é compatível com Apache, mas é mais rápido e eficiente com HTTPS.
Questionário 26: O que é o Microsoft IIS e em quais situações ele costuma ser adotado?

Resposta: O IIS é o servidor da Microsoft, ideal para ASP.NET e ambientes Windows.
Questionário 27: Explique para que serve o Node.js e em quais tipos de aplicações ele é mais utilizado.

Resposta: Node.js executa JavaScript no servidor, ideal para aplicações em tempo real.
Questionário 28: Descreva a função do servidor GlassFish e para qual linguagem ele é destinado.

Resposta: GlassFish é um servidor de aplicação Java EE para aplicativos corporativos.
Questionário 29: O que são Puma e Unicorn e em quais projetos eles são normalmente utilizados?

Resposta: Puma e Unicorn são servidores de aplicação para Ruby on Rails, usados em projetos web Ruby.
Questionário 30: Qual a importância da escolha adequada entre hospedagem local, compartilhada, VPS e nuvem no planejamento de um projeto web?

Resposta: A escolha certa de hospedagem (local, compartilhada, VPS, nuvem) impacta custo, desempenho e escalabilidade do projeto.
